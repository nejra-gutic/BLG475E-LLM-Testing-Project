# BLG475E - LLM-Based Code and Test Generation

Software Quality and Testing course project focused on evaluating Large Language Models (LLMs) for Java code and test generation using the HumanEval Java dataset.

## Project Goals

* Compare two LLMs for Java code generation
* Evaluate generated code using software testing techniques
* Analyze test quality, coverage, and correctness
* Perform black-box and white-box testing
* Assess effectiveness of LLM-generated tests

---

## Selected LLMs

* ChatGPT (GPT-4)
* Gemini

---

## Approach

Semi-agentic approach

In this approach, each step is manually managed by the team. The outputs generated by the LLMs are analyzed and used to guide the next steps such as code generation, testing, improvement, and refactoring.

---

## Repository Structure

### Phase1

* `code/` → Generated Java solutions from selected LLMs
* `tests/` → Base tests, improved tests, and black-box tests
* `logs/` → LLM interaction logs and prompt history
* `analysis/` → Coverage analysis and testing results
* `report/` → IEEE format reports
* `prompts/` → Selected HumanEval prompts and prompt analysis
